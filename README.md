# 🗂️ Desktop Organizer

바탕화면에 쌓여있는 파일들을 **확장자별로 자동 분류**해주는 Python 스크립트입니다.
어지러운 바탕화면을 보면서 "나중에 정리해야지..." 하다가 결국 안 하게 되어 만들었습니다.
이제 명령어 한 줄로 깔끔한 바탕화면을 만들어보세요! ✨

---

## 📋 주요 기능

### `list_files.py` - 파일 목록 미리보기
- 바탕화면의 모든 파일을 **카테고리별로 그룹화**하여 보여줍니다
- 각 파일의 **크기 정보**도 함께 표시합니다
- 파일을 이동하지 않고 **현재 상태만 확인**할 때 사용합니다

### `organize_desktop.py` - 자동 분류 실행
- 파일들을 확장자에 따라 **폴더로 자동 이동**합니다
- 이동 전 **미리보기**를 제공하고, 사용자 확인 후 실행합니다
- 선택적으로 **백업 폴더**를 생성할 수 있습니다

#### 📁 기본 분류 카테고리
| 카테고리 | 확장자 예시 |
|---------|------------|
| 이미지 | `.jpg`, `.png`, `.gif`, `.svg` 등 |
| 문서 | `.pdf`, `.docx`, `.xlsx`, `.hwp` 등 |
| 동영상 | `.mp4`, `.avi`, `.mkv`, `.mov` 등 |
| 음악 | `.mp3`, `.wav`, `.flac`, `.aac` 등 |
| 압축파일 | `.zip`, `.rar`, `.7z`, `.tar` 등 |
| 실행파일 | `.exe`, `.msi`, `.bat` 등 |
| 코드 | `.py`, `.js`, `.html`, `.css` 등 |
| CAD | `.dwg`, `.rvt`, `.ifc` 등 |
| 기타 | 위에 해당하지 않는 파일들 |

## 💻 설치 방법

### 1단계: Python 설치 확인

먼저 컴퓨터에 Python이 설치되어 있는지 확인합니다.

**Windows의 경우:**
- `Win + R` 키를 누르고 `cmd`를 입력하여 명령 프롬프트(터미널)를 엽니다
- 아래 명령어를 입력합니다:

```bash
python --version
```

`Python 3.x.x` 형태로 버전이 표시되면 설치되어 있는 것입니다.

> 💡 **Python이 없다면?**
> [python.org](https://www.python.org/downloads/)에서 Python 3.x 버전을 다운로드하여 설치하세요.
> 설치 시 **"Add Python to PATH"** 옵션을 꼭 체크해주세요!

> 💡 **"Add Python to PATH"가 뭔가요?**
> PATH는 컴퓨터가 프로그램을 찾는 경로 목록입니다.
> 이 옵션을 체크하면 어떤 폴더에서든 `python` 명령어를 사용할 수 있습니다.
> 체크하지 않으면 Python 설치 폴더의 전체 경로를 매번 입력해야 해서 불편합니다.

### 2단계: 프로젝트 다운로드

GitHub에서 프로젝트를 다운로드합니다.

**방법 A) Git이 설치된 경우:**
```bash
git clone https://github.com/gorila316-ux/desktop-organizer.git
```

**방법 B) Git이 없는 경우:**
- GitHub 페이지에서 `Code` → `Download ZIP` 클릭
- 원하는 위치에 압축 해제

### 3단계: 완료! 🎉

별도의 패키지 설치가 필요 없습니다!
이 프로젝트는 Python **표준 라이브러리**(Python 설치 시 기본 포함)만 사용합니다.

## 🚀 사용 방법

### 📍 실행 위치
스크립트를 실행하려면 먼저 프로젝트 폴더로 이동해야 합니다:
```bash
# Windows의 경우
cd C:\path\to\desktop-organizer

# 예시: 다운로드 폴더에 있다면
cd C:\Users\username\Downloads\desktop-organizer
```

**확인 방법:** `dir` (Windows) 또는 `ls` (Mac/Linux) 명령어로 `list_files.py`와 `organize_desktop.py`가 보이면 올바른 위치입니다.

### 파일 목록 확인하기 (list_files.py)

정리하기 전에 먼저 바탕화면에 어떤 파일들이 있는지 확인해보세요.

```bash
python list_files.py
```

**실행 결과 예시:**
```
Desktop: C:\Users\username\Desktop
Total files: 15 (125.3 MB)
==================================================

[DOCUMENTS] (5 files)
--------------------------------------------------
  보고서.pdf                            2.1 MB
  회의록.docx                           156.0 KB

[IMAGES] (8 files)
--------------------------------------------------
  스크린샷_001.png                      1.2 MB
  사진.jpg                              3.5 MB
```

### 바탕화면 정리하기 (organize_desktop.py)

실제로 파일을 폴더로 이동합니다.

```bash
python organize_desktop.py
```

**실행 흐름:**
1. 이동될 파일 목록을 미리 보여줍니다
2. `위 파일들을 이동하시겠습니까? (y/n):` → `y` 입력
3. `백업을 생성하시겠습니까? (y/n):` → 원하면 `y` 입력
4. 파일들이 자동으로 분류됩니다!

**정리 후 바탕화면 구조:**
```
바탕화면/
├── 이미지/
│   ├── 스크린샷_001.png
│   └── 사진.jpg
├── 문서/
│   ├── 보고서.pdf
│   └── 회의록.docx
├── 동영상/
│   └── 강의.mp4
└── _backup_20240129_143052/   ← 백업 선택 시 생성
```

### 카테고리 수정하기 (CATEGORY_MAP)

기본 분류 기준을 변경하고 싶다면 스크립트를 직접 수정할 수 있습니다.

1. VSCode(또는 메모장)로 `organize_desktop.py` 파일을 엽니다
2. 상단의 `CATEGORY_MAP` 부분을 찾습니다:

```python
CATEGORY_MAP = {
    "이미지": {".jpg", ".jpeg", ".png", ".gif", ...},
    "문서": {".pdf", ".doc", ".docx", ...},
    # ... 더 많은 카테고리
}
```

3. 원하는 대로 수정합니다:

```python
# 예시: 새 카테고리 추가
CATEGORY_MAP = {
    # ... 기존 카테고리 ...
    "3D모델": {".obj", ".fbx", ".stl", ".blend"},  # 새로 추가!
}
```

4. 파일을 저장하고 다시 실행합니다

## ⚠️ 주의사항

### 🔒 중요 파일은 백업하세요
- 정리 실행 전 **중요한 파일은 미리 다른 곳에 복사**해두는 것을 권장합니다
- 스크립트 실행 시 백업 옵션(`y`)을 선택하면 `_backup_날짜_시간` 폴더가 생성됩니다

### 👀 먼저 미리보기로 확인하세요
- **`list_files.py`를 먼저 실행**하여 어떤 파일들이 정리될지 확인하세요
- `organize_desktop.py`도 실제 이동 전에 미리보기를 보여주니 꼭 확인 후 진행하세요

### 📌 자동으로 건너뛰는 파일
- 바로가기 파일 (`.lnk`, `.url`)
- 시스템 파일 (`desktop.ini`, `thumbs.db`)
- 폴더 (파일만 정리합니다)

### 💡 유용한 팁
- 정기적으로 바탕화면을 정리하면 파일 관리가 쉬워집니다
- 중요 파일은 바탕화면 대신 전용 폴더에 보관하세요
